# oscope64

an interactive "deconstructed oscilloscope" visualizer for sm64 ds tracks. click play to start, click cells to mute/unmute channels, and pick songs from the dropdown.

inspired by [mr. oscilloscope's video](https://www.youtube.com/watch?v=-QmYJHMtn3Y). 

fun songs to try: athretic, mugen, vocal.

built with web audio api
