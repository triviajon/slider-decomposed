oscope64

hey! this is an interactive "deconstructed oscillicope" visualizer of tracks from sm64 (ds edition)

controls

click on the play button to start/stop the music

click on channels to individually mute/unmute it

select a new song in the top left corner

notes

this is inspired by this video by mr. oscilloscope

the code is available on github

fun songs to try are: athretic, mugen, and vocal
